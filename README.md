# Game Pigeon Solver

A web-based tool for solving popular **GamePigeon** word games such as **Anagrams** and **Word Hunt**.

## Overview
This project helps users quickly find all possible valid words for supported GamePigeon games using a clean, intuitive interface.

## Technologies Used
- TypeScript
- HTML/CSS
- Tailwind
- Vite

### Anagrams
- Input a set of letters as a word 
- Generates **all possible words** using a dictionary-based approach
- Results are organized by word length (e.g., 7-letter words, 6-letter words, etc.)
- Ordered from longest to shortest

### Word Hunt
- Select a matrix size
- Fill in the letter grid
- Displays all valid words that can be formed following Word Hunt rules

## Purpose
Built to practice algorithmic problem-solving, front-end development, and UI design while working with real-world game logic.
